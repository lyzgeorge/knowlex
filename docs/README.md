## 🏗 项目架构

```
Knowlex Desktop
├── 前端 (React + TypeScript)
│   ├── UI 组件 (Chakra UI)
│   ├── 状态管理 (Zustand)
│   └── 样式系统 (Emotion)
├── 后端 (Electron Main Process)
│   ├── 数据库服务 (SQLite)
│   ├── AI 服务 (OpenAI)
│   └── 文件处理 (Worker Threads)
├── 通信层 (IPC)
│   ├── 类型安全接口
│   ├── 流式数据支持
│   └── 错误处理机制
└── 共享类型 (TypeScript)
    ├── 数据库实体
    ├── API 接口
    └── 业务逻辑类型
```
